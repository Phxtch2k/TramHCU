# 🚋 Where’s My Tram? - Real-time Tracking System

**ระบบติดตามรถรางรับ-ส่ง** ภายในมหาวิทยาลัยแบบเรียลไทม์ แสดงผลผ่านเว็บแอปพลิเคชันที่ทันสมัย พร้อมระบบคำนวณเวลาถึงสถานีถัดไป (ETA) โดยใช้ข้อมูลจาก **Firebase Real-time Database**

---

## 🌟 คุณสมบัติหลัก (Features)

* **Real-time Tracking:** ติดตามตำแหน่งรถรางบนแผนที่จำลองได้ทันทีเมื่อรถผ่านจุดตรวจเช็ก
* **Smart ETA Countdown:** ระบบคำนวณเวลาที่คาดว่าจะถึงสถานีถัดไป โดยวิเคราะห์จากสถิติเวลาเดินทางจริง และนับถอยหลังแบบเรียลไทม์
* **Adaptive Learning:** ระบบจดจำระยะเวลาเดินทางในแต่ละช่วงสถานีอัตโนมัติ เพื่อความแม่นยำที่สูงขึ้นเรื่อยๆ
* **Historical Logs:** แสดงรายการสถานีล่าสุด พร้อมเวลาที่บันทึกจริงในรูปแบบ **พ.ศ.**
* **Time-Sync Logic:** ระบบนับถอยหลังอ้างอิงจากเวลาเครื่อง (System Time) ทำให้รีเฟรชหน้าจอแล้วเวลาไม่เพี้ยน

---

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend:** Firebase Real-time Database
* **Hardware:** ESP32 / RFID Reader
* **Hardware:** Bluetooth Low Energy (BLE) Beacons, ESP32 (BLE Scanner mode)
* **Tracking Logic:** ใช้การตรวจสอบความแรงสัญญาณ (RSSI Threshold) เพื่อระบุตำแหน่งสถานี
---

## 💡 ระบบคำนวณเวลา (ETA Logic)

ระบบใช้หลักการ **(Predicted Time - Elapsed Time)**:
* **Predicted Time:** เวลาเฉลี่ยที่รถวิ่งระหว่างสถานี (ดึงจากประวัติ)
* **Elapsed Time:** เวลาที่ผ่านไปนับจากที่เจอรถครั้งสุดท้าย (คำนวณจากนาฬิกาเครื่อง)
* **Result:** ทำให้ได้เวลานับถอยหลังที่แม่นยำ แม้จะมีการรีเฟรชหน้าจอก็ตาม
